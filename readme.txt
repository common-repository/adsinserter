Stable tag: trunk

Manage your site placements from [adsinserter.com dashboard](https://adsinserter.com).
Show placement units by filters: device type, referrer, url, tags and more.

Plugin:
* Connect client script and configure tags on your page.
* Add widget for install placement from admin UI to widget area
* Auto in text placement insertion with custom density

= Features =
* Target units by:
 * Device type
 * Screen size
 * Url or referrer substring
 * Tags
 * Countries
* Perfomance
 * Does not affect the performance of your server
 * Works with all types of caching and with all cache plugins
 * Lazyload option
* Rorate units in placement
* Starting and stopping units on schedule
* Auto stop units after N views
* Views statistics
* Share access to the project and configure privileges
* Change log

= How to start =
1. Install and activate plugin on your wordpress site
1. Sign in to [adsinserter.com dashboard](https://adsinserter.com/login)
2. Create project for your site
3. Create placement and insert code to your site where you want see ads or widgets
4. Create unit in new placement with ads or widget code, configure settings and filters (if needed). Save unit
